## Run Locally

Clone the project

```bash
  git clone https://github.com/SoumyashreeBehera/my-first-react-app.git
```

Go to the project directory

```bash
  cd my-first-react-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
